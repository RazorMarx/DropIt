# DropIt
One php file fancy file uploader

![image](https://github.com/RazorMarx/DropIt/assets/16266220/7add8ea6-e286-4b4d-9384-12e135252adb)

# Feature :
- Drag'n'Drop files
- Multiples files
- Big file uploading (sliced and recomposed)
- Too much animation
